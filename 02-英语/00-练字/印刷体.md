# 练字准备
- 四线三格本
- 笔
# 练字步骤
- 每天认真写2-3行句子；
- 红笔订正修改。
# 字迹规范
- 倾斜10°-15°；
- 每个字母尽量写得饱满、圆润；
- 不要添笔、描改；
- 每段前空大约4个字母的宽度；
- 单词之间间隔 0.5-1 个字母宽度。
- 上不接天，下不接地：字母最顶部不超过四线三格最上面一格的$\frac{1}{2}\text{至}\frac{3}{4}$处，字母最底部不超过四线三格最下面一个的$\frac{1}{2}\text{至}\frac{3}{4}$处。
