# 注意点
1. 人名直接音译。
2. 时间地点可以翻译到一起。
	- [20世纪60年代]的[西欧]
3. 尽量把被动翻译为主动
4. 名词的定语从句 -> ......【名词】，这些【名词】【定语从句】。
	- 附录列出来许多【加密文件】，这些【文件】【缴获自西班牙法军】。
5. 动词的宾语从句 -> 【宾语从句】，【主语】对此【谓语】。
	- 【在这场多个国家参加的大型战争中，这位名不见经传的军官可能层做出过贡献，也可能不曾】，【欧曼】对此【未能做出过仔细的分析】。
6. It is [adj.] to do sth -> 【做什么事】，"是"【如何的】。
	- 但想要找到真正出自他们之手或由他们经手的，却往往极其困难。
7. Just as ..., so ... -> 正如......，......也一样。
8. speak volumes about: to provide a lot of information about something / to show sth very clearly -> 清楚地说明
9. attempt -> 表达出想要的意思就行
10. 翻译尽量不要写“以致于”。
11. 多个动作在一句话，切分为两句话。
12. 小语种不用翻译，直接抄写即可。如：拉丁语
