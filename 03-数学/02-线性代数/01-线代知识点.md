# 第一章 行列式
- [ ] 对于两个 $n$ 阶方阵 $A$ 和 $B$，公式 $|AB| = |A| \cdot |B|$ 总是成立吗？
	对
- [ ] 如果 $A$ 是 $m \times n$， $B$ 是 $n \times m$，且 $m > n$（即“高瘦”乘“矮胖”），生成的 $m$ 阶方阵 $AB$ 的行列式是多少？
	0
	- **中间窄**：中间的公共维度 $n$ 小于最终维度 $m$。
    - **秩受限**：根据秩的不等式 $r(AB) \le r(A) \le \min(m, n) = n$。
    - **结论**：因为 $r(AB) \le n < m$，所以 $AB$ 不满秩，行列式必为 0。
    - _形象理解：中间通道太窄，把 $m$ 维的体积压扁了，体积（行列式）归零。_

# 第三章 n维向量
- [ ] 判断向量组$\alpha_{1},\alpha_{2},\dots,\alpha _{n}$的线性相关性【P68】
	1. 根据定义
		- 若存在不全为零的常数$k_{1},k_{2},\dots,k_{n}$，使得$k_{1}\alpha_{1}+k_{2}\alpha_{2}+\dots+k_{n}\alpha_{n}=0$，则称向量组$\alpha_{1},\alpha_{2},\dots,\alpha _{n}$线性相关；
		- 若由$k_{1}\alpha_{1}+k_{2}\alpha_{2}+\dots+k_{n}\alpha_{n}=0$一定能推出$k_{1}=k_{2}=\dots=k_{n}=0$，则称向量组$\alpha_{1},\alpha_{2},\dots,\alpha _{n}$线性无关。
	2. 计算$r(\alpha_{1},\alpha_{2},\dots,\alpha _{n})$
		- 若$r(\alpha_{1},\alpha_{2},\dots,\alpha _{n})<n$，则向量组$\alpha_{1},\alpha_{2},\dots,\alpha _{n}$线性相关；
		- 若$r(\alpha_{1},\alpha_{2},\dots,\alpha _{n})=n$，则向量组$\alpha_{1},\alpha_{2},\dots,\alpha _{n}$线性无关。
	3. 考虑举反例，例如 $\boldsymbol{\overrightarrow{0}}$
- [ ] 线性表出的定义是什么？【P76】
	对$n$维向量$\alpha_{1},\alpha_{2},\dots,\alpha_{s}$和$\beta$，若存在实数$k_{1},k_{2},\dots,k_{s}$使得$$k_{1}\alpha_{1}+k_{2}\alpha_{2}+\dots+k_{s}\alpha_{s}=\beta，$$则称$\beta$是$\alpha_{1},\alpha_{2},\dots,\alpha_{s}$的线性组合，或者说$\beta$可由$\alpha_{1},\alpha_{2},\dots,\alpha_{s}$线性表出。

# 第五章 特征值与特征向量
- [ ] $n$阶矩阵$A$，$B$相似，说明什么？（相似的必要条件）【P140】
	- 它们的特征多项式、特征值、秩、行列式、迹相同。
	- $A+kE \sim B+kE,(A+kE)^{n} \sim (B+kE)^{ n},f(A)\sim f(B)$
	- $A^{T}\sim B^{T},A^{*}\sim B^{*},A^{-1}\sim B^{-1}(若A可逆)$
	- 特征值相同，且相同特征值所对应的线性无关的特征向量的个数相同。
- [ ] $n$阶矩阵相似对角化的判定条件是什么？【P109】
	- 充要条件：
		- $A$有$n$个线性无关的特征向量。
		- $A$的每个特征值中，线性无关的特征向量的个数恰好等于该特征值的重数。
		$A \sim \Lambda \iff \lambda_{i}是A的n_{i}重特征值，则\lambda_{i}有n_{i}个线性无关的特征向量$。
		$\iff 秩r(\lambda_{i}E-A)=n-n_{i}，\lambda_{i}为n_{i}重特征值$
	- 充分条件：
		- $A$有$n$个不同的特征值；
		- $A$为实对称矩阵。

# 第六章 二次型
- [ ] 怎么用配方法化没有平方项的二次型为标准型？
	- 利用平方差公式进行可逆线性变换。
		1. **选取交叉项**：在二次型中任选一个系不为 0 的交叉项，例如 $a_{12}x_1x_2$（假设系数 $a_{12} \neq 0$）。   
		2. 实施三角代换（创造平方项）： 
			令：
			$$\begin{cases} x_1 = y_1 + y_2 \\ x_2 = y_1 - y_2 \\ x_3 = y_3 \\ \vdots \\ x_n = y_n \end{cases}$$
		    _注：对于不涉及的变量（如 $x_3, \dots, x_n$），直接保持不变即可。这一步变换的可逆性是有保证的（行列式不为0）。_    
		3. 代入并整理：		    
		    将上述关系代入原二次型 $f(x_1, \dots, x_n)$ 中。		    
		    此时，原式中的 $a_{12}x_1x_2$ 这一项就会变成：	    
		    $$a_{12}(y_1 + y_2)(y_1 - y_2) = a_{12}y_1^2 - a_{12}y_2^2$$
		    
		    这就成功制造出了 $y_1^2$ 和 $y_2^2$ 两个平方项。		    
		4. 常规配方：		    
		    现在表达式中已经有了平方项（$y_1^2$ 或 $y_2^2$），接下来就可以按照常规的配方法（“选取一个平方项，将含有该变量的所有项括在一起配成完全平方”）继续操作，直到化为标准型。
## 正定二次型
- [ ] 正定二次型、正定矩阵的定义是什么？
	$n$元二次型$f(x_{1},x_{2},\dots,x_{n})=x^{T}Ax$，若对任意的$x=[x_{1},x_{2},\dots,x_{n}]^{T}\neq 0$，均有$x^{T}Ax>0$，则称$f$为正定二次型，称二次型对应的矩阵$A$为正定矩阵。
	<mark style="background: #D2B3FFA6;">前提</mark>：$A=A^{T}$（$A$是是对称矩阵）
- [ ] 二次型正定的充要条件有哪些？
	$$\begin{aligned}n元二次型f=x^{T}Ax正定 &\Leftrightarrow 对任意x\neq 0，均有x^{T}Ax>0 \\&\Leftrightarrow A的特征值\lambda_{i}>0(i=1,2,\dots,n)\\&\Leftrightarrow f的正惯性指数p=n\\&\Leftrightarrow A的全部顺序主子式均大于0\\&\Leftrightarrow 存在可逆矩阵D，使A=D^{T}D\\&\Leftrightarrow A与E合同\end{aligned}$$
- [ ] 二次型正定的必要条件有哪些？
	1. $a_{ii}>0(i=1,2,\dots,n)$
	2. $|A|>0$
	3. 若$A$是正定矩阵，则$A^*$与$A^{-1}$也是正定矩阵

	2. 特征值：

	3. 幂等性：
		$(\alpha\alpha^{T})^{n}=\alpha\alpha^{T}$
