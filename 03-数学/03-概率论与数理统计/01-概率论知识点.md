# 第二章 一维随机变量及其分布
- [ ] 写分布函数时，范围用左闭右开还是左开右闭？【P144】
	左闭右开
- [ ] 泊松分布的概率分布和期望、方差是什么？【P141】
	- $P\{X=k\}= \frac{\lambda^{k}e^{ -\lambda }}{k!},k=0,1,2,\dots$，其中$\lambda>0$
	- $E(X)=\lambda,D(X)=\lambda$
- [ ] 均匀分布的概率密度和分布函数是什么？期望和方差又是多少？
	- 概率密度：
		- $$f(x)=\begin{cases}  \frac{1}{b-a},\ a\leq x\leq b\\  \\0,\ 其他\end{cases} $$
	- 分布函数：
		- $$F(x)=\begin{cases} 0,\ x<a,\\ \frac{x-a}{b-a},\ a\leq x\leq b\\1,x\geq b\end{cases} $$
	- 均匀分布的期望$E(X)=\frac{a+b}{2}$，方差$D(X)=\frac{(b-a)^{2}}{12}$。
- [ ] 指数分布的概率密度和分布函数是什么？期望和方差又是多少？它的无记忆性指的是什么？
	- 概率密度：
		- $$f(x)=\begin{cases}  \lambda e^{-\lambda x},x\gt 0\\  \\0,x\leq 0\end{cases} $$
	- 分布函数：
		- $$F(x)=\begin{cases}  1- e^{-\lambda x},x\gt 0\\  \\0,x\leq 0\end{cases} $$
	- 指数分布的期望$E(X)=\frac{1}{\lambda}$，方差$D(X)=\frac{1}{\lambda^2}$。
	- 无记忆性： $P\{X>s+t|X>s\}=P\{X>t\},\forall s,t>0$
- [ ] 正态分布如何进行标准化？
	若随机变量$X\sim N(\mu,\sigma^{2})$，则$Z=\frac{X-\mu}{\sigma}\sim N(0,1)$。

- [ ] 协方差的性质有哪些？【P196】
	- $cov(X,Y)=cov(Y,X)$
	- $cov(X,X)=D(X)$
	- $cov(C,X)=0，其中\ C\ 为任意常数$
	- $cov(aX,bY)=abcov(X,Y)，其中\ a,b\ 为任意常数$
	- $cov(k_{1}X_{1}\pm k_{2}X_{2},Y)=k_{1}cov(X_{1},Y)\pm k_{2}cov(X_{2},Y)$
- [ ] 相关系数的定义是什么？【P197】

# 第五章 大数定律和中心极限定理
- [ ] 切比雪夫不等式是什么？【P206】
	设随机变量$X$具有数学期望$E(X)=\mu$，方差$D(x)=\sigma^{2}$，则对任意$\epsilon>0$，均有
	$$P\{|X-E(X)|\geq\epsilon\}\leq \frac{D(x)}{\epsilon^{2}}或P\{|X-E(X)|<\epsilon\}\geq 1- \frac{D(x)}{\epsilon^{2}}$$
- [ ] 辛钦大数定理是什么？【P208】
	设随机变量$X_{1},X_{2},\dots ,X_{n}$相互独立，服从相同的分布，具有数学期望$E(X_{n})=\mu,n=1,2,\dots,n$，则对于任意给定的正数$\epsilon$，总有
	$$\lim_{ n \to \infty }P\left\{ \begin{vmatrix}\frac{1}{n}\sum_{k=1}^{n}X_{k}-\mu\end{vmatrix} <\epsilon \right\}=1或 \lim_{ n \to \infty }P\left\{ \begin{vmatrix}\frac{1}{n}\sum_{k=1}^{n}X_{k}-\mu\end{vmatrix} \geq \epsilon \right\}=0$$
	
- [ ] 列维-林德伯格中心极限定理和棣莫弗-拉普拉斯中心极限定理是什么？【P209】
	- 列维-林德伯格中心极限定理：
		设随机变量$X_{1},X_{2},\dots ,X_{n},\dots$相互独立，服从相同的分布，具有数学期望$E(X_{n})=\mu$和方差$D(X)=\sigma^{2}$，$n=1,2,\dots$，则对于任意实数$x$，有$\lim_{ n \to \infty }P\left\{ \frac{\sum_{k=1}^{n}X_{k}-n\mu}{\sqrt{ n }\sigma}\leq x  \right\}= \Phi(x)$。
	- 棣莫弗-拉普拉斯中心极限定理：
		设随机变量$X_{n}$服从参数为$n，p$的二项分布，即$X\sim B(n,p)，0<p<1,n=1,2,\dots$，则对于任意实数$x$，有$\lim_{ n \to \infty }P\left\{ \frac{X_n-np}{\sqrt{ np(1-p) }}\leq x  \right\}= \Phi(x)$。

### （一）$\chi^{2}$分布
- [ ] $\chi^{2}$分布的典型模式、性质、图像、数字特征是什么？
1. 典型模式
	设随机变量$X_{1},X_{2},\dots,X_{n}$相互独立，都服从标准正态分布$N(0,1)$，称随机变量$\chi^{2}=X_{1}^{2}+X_{2}^{2}+\dots+X_{n}^{2}$为服从自由度是$n$的$\chi$分布，记作$\chi^{2}\sim\chi^{2}(n)$。
2. 性质
	设$X\sim\chi^{2}(m)$，$Y\sim \chi^{2}(n)$，且$X$和$Y$相互独立，则$X+Y\sim\chi^{2}(m+n)$。

# 第七章 参数估计与假设检验
- [ ] 最大似然估计解题的一般步骤是什么？【P228】
	1. 写出似然函数：
		- $L(\theta)=L(x_{1},x_{2},\dots,x_{n};\theta)=\prod_{i=1}^{n}p(x_{i},\theta)$（离散型）
		- $L(\theta)=L(x_{1},x_{2},\dots,x_{n};\theta)=\prod_{i=1}^{n}f(x_{i},\theta)$（连续型）
	2. 判断$L(\theta)$是否为关于$\theta$的单调函数：
		- 若是，根据函数关系找出让$L(\theta)$最大的$\theta$；
		- 若不是，继续往下执行；
	3. 取对数$\ln L(\theta)$
	4. 将$\ln L(\theta)$对$\theta$求导$\frac{d\ln L(\theta)}{d\theta}$
	5. 判断方程组$\frac{d\ln L(\theta)}{d\theta}=0$是否有解，若有唯一的解，则其解即为所求最大似然估计；若有不同的解，则根据题干条件取舍；若无解，则最大似然估计常在$\theta$取值的端点上取得。
- [ ] 无偏性的定义是什么？【P234】
	如果估计量$\hat{\theta}(X_{1},X_{2},\dots,X_{n})$的数学期望$E(\hat{\theta})$存在，且对于任意$\theta \in \Theta$。有$E(\hat{\theta})=\theta$，则称$\hat{\theta}$是未知参数$\theta$的无偏估计量。
- [ ] 一致性（相合性）的定义是什么？【P235】
	设估计量$\hat{\theta}(X_{1},X_{2},\dots,X_{n})$为未知数$\hat{\theta}$的估计量，如果对于任意$\theta \in \Theta$。当$n\to \infty$时，$\hat{\theta}(X_{1},X_{2},\dots,X_{n})$依概率收敛与$\theta$，则称$\hat{\theta}$是未知参数$\theta$的一致估计量或相合估计量。
- [ ] 一个正态总体参数的区间估计是什么？（$1-\alpha$置信区间）【P236】
	- 未知参数$\mu$:
		- $\sigma^{2}$已知：
			- $\left( \overline{X}-U_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{ n }} ,\overline{X}+U_{\frac{\alpha}{2}} \frac{\sigma}{\sqrt{ n }} \right)$
			- $U= \frac{\overline{X}-\mu}{\frac{\sigma}{\sqrt{ n }}}\sim N(0,1)$
		- $\sigma^{2}$未知：
			- $\left( \overline{X}-t_{\frac{\alpha}{2}}(n-1) \frac{S}{\sqrt{ n }} ,\overline{X}+t_{\frac{\alpha}{2}}(n-1) \frac{S}{\sqrt{ n }} \right)$
			- $\frac{\overline{X}-\mu}{\frac{S}{\sqrt{ n }}}\sim t^{2}(n-1)$
	- 未知参数$\sigma^{2}$：
		- $\mu$已知：
			- $\left( \frac{\sum_{i=1}^{n}(X_{i}-\mu^{2})}{\chi^{2}_{\frac{\alpha}{2}}(n)},\frac{\sum_{i=1}^{n}(X_{i}-\mu^{2})}{\chi^{2}_{1-\frac{\alpha}{2}}(n)} \right)$
			- $\chi^{2}=\frac{1}{\sigma^{2}}\sum_{i=1}^{n}(X_{i}-\mu)^{2}\sim\chi^{2}(n)$
			
		- $\mu$未知：
			- $(\frac{(n-1)S^{2}}{\chi^{2}_{\frac{\alpha}{2}}(n-1)},\frac{(n-1)S^{2}}{\chi^{2}_{1-\frac{\alpha}{2}}(n-1)})$
			- $\frac{(n-1)S^{2}}{\sigma^{2}}\sim \chi^{2}(n-1)$
