# 具体原则
动态、连贯、具体
# 注意点
1. 人名直接音译。
2. 名词的定语从句 -> ......【名词】，这些【名词】【定语从句】。
	- 附录列出来许多【加密文件】，这些【文件】【缴获自西班牙法军】。
3. 动词的宾语从句 -> 【宾语从句】，【主语】对此【谓语】。
	- 【在这场多个国家参加的大型战争中，这位名不见经传的军官可能层做出过贡献，也可能不曾】，【欧曼】对此【未能做出过仔细的分析】。
4. It is [adj.] to do sth -> 【做什么事】，"是"【如何的】。
	- 但想要找到真正出自他们之手或由他们经手的，却往往极其困难。
5. Just as ..., so ... -> 正如......，......也一样。
6. speak volumes about: to provide a lot of information about something / to show sth very clearly -> 清楚地说明
